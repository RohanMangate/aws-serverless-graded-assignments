# AWS Serverless Graded Assignments

This repository contains Python-based AWS Lambda implementations developed as part of the **Serverless Architecture and Cloud Automation** graded assignments.

The assignments demonstrate automation and event-driven design using AWS services such as **Amazon S3, DynamoDB, SNS, and EventBridge**, with **Boto3** used as the primary SDK for interacting with AWS resources programmatically.

## Assignments Covered

- **Assignment 2:** Automated S3 Bucket Cleanup using AWS Lambda and Boto3  
- **Assignment 3:** Monitoring Unencrypted S3 Buckets using AWS Lambda  
- **Assignment 7:** DynamoDB Item Change Alerts using Lambda and SNS  
- **Assignment 15:** Automated Log Cleaner for S3 using Lambda and EventBridge  

Each folder in this repository corresponds to a specific assignment and contains the respective Lambda function source code.

## Notes

- All Lambda functions were manually implemented and tested using the AWS Management Console.
- Execution and validation were performed using Amazon CloudWatch logs.
- Detailed step-by-step documentation and screenshots are provided separately as part of the assignment submission.
